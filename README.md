# AI Agent Lab

AI Agent 全栈实验项目 - 探索 skills、memory、tool use 等 agentic 核心能力。

## 项目目标

- 学习并实践 Agent 开发的核心架构模式
- 构建可独立交付的 AI Agent 产品
- 积累 Agent 开发作品集

## 技术栈

- **Agent SDK**: Claude Agent SDK (TypeScript / Python)
- **前端**: React / Next.js
- **后端**: Node.js / Python
- **核心能力**: Skills System, Memory Management, Tool Use, Multi-Agent

## 项目结构

```
ai-agent-lab/
├── experiments/       # 各类 Agent 实验
├── skills/            # 自定义 Skills
├── memory/            # Memory 系统实现
├── tools/             # Tool Use 实现
├── agents/            # 完整 Agent 应用
└── docs/              # 学习笔记与文档
```

## 学习路线

1. **Phase 1**: Claude API + Prompt Engineering 基础
2. **Phase 2**: Tool Use + Skills 系统开发
3. **Phase 3**: Memory 机制与持久化
4. **Phase 4**: Multi-Agent 协作
5. **Phase 5**: 完整产品交付
